## Pong Game

### About: 
  This game was developed during the game development course at NoNe Class. Despite being a simple project and having been done with the professor, I was happy with the result because, for someone who knows nothing about programming, finishing the project was gratifying.

### Controls:
  ### Player 1's Paddle
    W = Up
    S = Down
  ### Player 2's Paddle
    Up Arrow = Up
    Down Arrow = Down
  For player 2 to join the game, just press any of the arrow keys, and the player will take control of the paddle that was previously controlled automatically. To make the paddle return to being controlled automatically, just press the 'Enter' button.
